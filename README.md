## 部分依赖

博客文章使用的Markdown格式来编写和展示。

[editor.md](https://github.com/pandao/editor.md)库提供对markdown的支持。

[oidc-client](https://github.com/IdentityModel/oidc-client-js)库提供对博客后台API接口的OAuth2.0授权认证操作。
